多级反馈调度队列可视化

![image](https://github.com/Chilly-Blaze/MLFQ-visualization/assets/74091261/0ff9a246-b563-45c6-977e-1fcfb1961a62)


## 项目安装

1. 下载pnpm，配置软件源
2. `git clone https://github.com/Chilly-Blaze/MLFQ-visualization.git`
3. 进入项目目录
4. `pnpm install`

### 本地编译

```sh
pnpm dev
```

### 打包

```sh
pnpm build
```
